# ヘルプドキュメント

このディレクトリには、ヘルプページで表示されるMarkdownファイルが格納されています。

## 編集方法

このディレクトリ内のファイルを編集してください。

## 自動生成について

編集したファイルは、`npm run build`実行時に自動的に`public/content/help/`にコピーされます。

- **開発時**: このディレクトリ（`src/content/help/`）が直接参照されます
- **ビルド時**: `public/content/help/`にコピーされたファイルが参照されます

## 注意事項

- `public/content/help/`内のファイルは**自動生成**されるため、直接編集しないでください
- 常にこのディレクトリ（`src/content/help/`）のファイルを編集してください
